# Time Travel Web App

A simple web application to display the current time in your location and other selected locations. Locations are persisted using localStorage.

## Features

- Set your current time zone
- Add multiple locations to compare times
- Times update in real-time
- Persistent storage of locations

## Usage

1. Open `index.html` in a web browser.
2. Enter your current time zone (e.g., Europe/Prague) and click "Set Current".
3. Add other locations by entering their time zones and clicking "Add".
4. View the times updating every second.
5. Locations are saved and will be loaded on page reload.

## Time Zone Format

Use IANA time zone names, such as:
- America/New_York
- Europe/London
- Asia/Tokyo
- Australia/Sydney
